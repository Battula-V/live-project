Creating Multiple projects in only one angular application

It is not good idea to create different angular application for different projects. the best idea is to create multiple projects in one angular application

To achieve this create folder called modules and create module for every project

What is module in angular

A module is a group of services,components and other modules

Command for creating a module

ng generate module moduleName (ng g m moduleName)

Now create a component with same module name

ng generate component moduleName

Create routing module with same module name

ng generate module moduleName-routing

Note:- make sure component , routing and module files must be under same folder
