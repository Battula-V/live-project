Sharing data using url

Data can be shared from one component to another component using url. It has two ways

  1. using parameter (starts with slash(/) in Url )
  2. using query string (starts with question mark(?) in url)

1.using parameter (starts with slash(/) in Url )

In routing add :parameterName in path after the route. If route is abc and parameterName is salary , then routing will be like below 

 {{code1}}

In any component create an link and add the component route with parameter value like below

{{code2}}

Now to get the salary parameter value 1200 into the ts-file follow the below process

  1. import ActivatedRoute
  2. create ActivatedRoute object in the constructor
  3. Now subscribe to params property in the ActivatedRoute class
  4. Now it will show the value of a salary
