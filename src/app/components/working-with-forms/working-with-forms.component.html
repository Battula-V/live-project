Sending Data From Html To TS File

We can Send the data from html to ts-file with the help of forms

Working with forms

There are two types forms in angular

  1. Template Driven forms (Basic)
  2. Reactive Forms (Advanced)

Template Driven forms (Basic)

import FormsModule in module file like this:- import FormsModule from '@angular/forms';

Here , property binding , using ngModel attribute , is used for sending data from html to ts-file and data from ts-file to html file. This is called "Two-way binding". and this can be achieved by ngModel . This is applicable only for the form elements

Syntax:- In Html 
{{code1}}
In Ts-file

variableName = value;

Reactive Forms (Advanced)

import ReactiveFormsModule in module file like this:- import ReactiveFormsModule from '@angular/forms';

Here , property binding , using formControl attribute , is used for sending data from html to ts-file and data from ts-file to html file. This is called "Two-way binding". and this can be achieved by formControl . This is applicable only for the form elements

Syntax:- 
{{code2}}
In Ts-file

import FormControl, Validators from '@angular/forms';

variableName = new FormControl(value);
