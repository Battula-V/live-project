What are angular life-cycle hooks

Every page we are seeing on the browser is a component in angular. To understand the actions performing from opening the page closing the page , we need some pre-defined functions to know which action is performing

So angular is providing some pre-defined functions to know the actions performing on the page(or)component. This are called angular life-cycle hooks.

  1. ngOnInit
  2. ngDoCheck
  3. ngAfterContentInit
  4. ngAfterContentChecked
  5. ngAfterViewInit
  6. ngAfterViewChecked
  7. ngOnChanges
  8. ngOnDestroy
