Array Variable Definition

Storing many values in one variable is called an "Array variable"

syntax: variableName:Array < dataType > = [value1, value2, ........., value-n];

Example: myFriends:Array < string > = ["somu", "ramu"];

{{myFriends}}

The above Array variable displayed all values in the array

but if you want display any one value in the array, we have to do in the following way

syntax: variableName[index]

Index means position (or) serial number of the value in the array.

position (or) serial number start from 0(zero).

In the above example somu position is 0 and ramu position is 1. So if you want display ramu the variable has to written like below

myFriends[1]

{{myFriends[1]}}

Note

If you want store any data type in the array, in place of data type we can write any

{{friends}}

Any one value in array

{{friends[2]}}
