Combinator Selectors:
===================
1) Descendant Selector(Space)
2) Child Selector(>)
3) Adjacent Sibling Selector(+)
4) General Siblings Selector(~) 

1) Descendant Selector:
====================
This selector selects all specified descendant elements of 
specified element.

Example:
========
.courses h1
{
	color: red;
}

2) Child Selector:
===============
This selector selects direct child elements of specified 
element.

Example:
========
.courses > h1
{
	color: red;
}

3) Adjacent Sibling Selector:
========================
This selector selects adjacent sibling element of specified
element.

Example:
========
.courses + h1
{
	color: red;
}

4) General Siblings Selector:
========================
This selector selects all sibling elements of specified
element.

Example:
========
.courses ~ h1
{
	color: red;
}
