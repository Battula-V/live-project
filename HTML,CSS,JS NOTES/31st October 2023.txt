Text Properties:
==============
1) color => It is used to set the text color.
2) text-align => It is used to specify horizontal alignment.
Possible values are left, right, center & justify.
3) vertical-align => It is used to specify the superscripted text
		or subscripted text.
Possible values are sub & super.
4) letter-spacing => It is used to specify the space between
letters.
5) word-spacing => It is used to specify the space between
words.
6) line-height => It is used to specify the height of the line.
		(Space between lines).
7) text-decoration-line => It is used to specify that the line
is underline, overline or line-through.
8) text-decoration-color => It is used to specify the line color.
9) text-decoration-style => It is used to specify the line style.
Possible values are solid, dashed, dotted, double & none.
10) text-decoration-thickness => It is used to specify the line
thickness.
11) text-decoration => It is used to specify all values that are
related to line.
12) text-shadow => It is used to specify different types of
shadows.
