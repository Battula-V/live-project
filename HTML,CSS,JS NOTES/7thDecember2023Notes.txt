3) history object:
==============
It refers browser's history.

Methods of history object:
======================
1) back() => It is used to get the backward page which is 
already visited.
2) forward() => It is used to get the forward page which is
already visited.

Date object
==========
It is used to get the system date & time.

Methods of Date object:
====================
1) getDate() => It is used to get the date.
2) getMonth() => It is used to get the month.
3) getFullYear() => It is used to get the year.
4) getHours() => It is used to get the hours.
5) getMinutes() => It is used to get the minutes.
6) getSeconds() => It is used to get the seconds.
7) toString() => It is used to get the string representation of
date & time.
