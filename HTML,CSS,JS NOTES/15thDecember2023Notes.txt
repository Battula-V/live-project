Polymorphism:
=============
Poly means many, morphism means forms & 
polymorphism means many forms.
		(or)
The ability to take more than one form is called as
polymorphism.

<!-- Program to demonstrate Inheritance & Polymorphism -->
<html>
<body>
<script>
class Student
{
	rollNo=1;
	name="aaa";
	display()
	{
		document.write(this.rollNo, this.name);
	}
}
class Student2 extends Student
{
	address="Hyderabad";
	display()
	{
		super.display();
		document.write(this.address);
	}
}
let s=new Student2();
s.display();
</script>
</body>
</html>

In the above example,
Student class inherited in Student2 class.
display() method having two forms. It is called
polymorphism.
super keyword used to access super class member
whenever super class member & sub class member
names are same.
