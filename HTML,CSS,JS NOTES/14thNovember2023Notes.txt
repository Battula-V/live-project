<!-- Program to demonstrate bigint literal -->
<html>
<body>
<script>
let a=999999999999999;
let b=9999999999999999;
let c=9999999999999999n;
document.write(a+"<br>");
document.write(b+"<br>");
document.write(c);
</script>
</body>
</html>

If the number is having more than 15 digits then use bigint
literal.
Bigint literal must be suffixed with n.

<!-- Program to demonstrate string literals -->
<html>
<body>
<script>
let s1='welcome';
let s2="welcome";
let s3=`welcome`;
let s4=`Welcome to 
			Naresh IT`;
document.write(s1+"<br>");
document.write(s2+"<br>");
document.write(s3+"<br>");
document.write(s4);
</script>
</body>
</html>

Note: Multi line string literal must be in backticks only.

typeof operator:
==============
It is used to get the type of data. 
Here type decided dynamically whenever variable initialized.
typeof operator can be used as follows:
typeof(variable) (or) typeof variable
