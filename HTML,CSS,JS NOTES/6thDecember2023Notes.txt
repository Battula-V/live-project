BOM:
=====
BOM stands for Browser Object Model.
It contains objects that allows Java Script to interact
with browser.

List of objects:
=============
1) window object
2) screen object
3) history object

1) window object:
===============
It represents a browser window.

Properties of window object:
========================
1) innerWidth => It is used to get the window width.
2) innerHeight => It is used to get the window height.

Example:
========
<html>
<body>
<script>
document.write("Window Width: "+window.innerWidth+"Pixels<br>");
document.write("Window Height: "+window.innerHeight+"Pixels");
</script>
</body>
</html>

Methods of window object:
======================
1) open() => It is used to open a new window.
2) close() => It is used to close a specified window.
3) alert() => It is used to display alert pop up window.
4) confirm() => It is used to display confirm pop up window.
5) prompt() => It is used to display prompt pop up window.
6) setTimeout() => It is used to specify time to start the task.
7) setInterval() => It is used to specify the time interval to 
repeat the task.

<!-- Program to demonstrate open() & close() methods -->
<html>
<body>
<input type="button" value="Open" onclick="fun1()">

<body>

<!-- Program to demonstrate prompt() method -->
<html>
<body>
<script>

{
	document.write("You entered negative number");

	document.write("You entered positive number");
}
else
{
	document.write("You entered zero");

</body>
</html>

2) screen object:
==============
It represents user's screen.

Properties of screen object:
=======================
1) width => It is used to get the screen width.
2) height => It is used to get the screen height.

Example:
========
<html>
<body>
<script>
document.write("Screen Width: "+screen.width+"Pixels<br>");
document.write("Screen Height: "+screen.height+"Pixels");
</script>
</body>
</html>
