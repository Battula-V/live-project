<h2>HTML</h2>
<p>HTML stands for Hyper Text Markup Language. It is called as presentation language. It is also called as formatting language. It is used to create a web page. It was developed by Tim Berners Lee in 1993.</p>
<h2>CSS</h2>
<p>CSS stands for Cascading Style Sheets. It is called as style sheet language. It is used to define style sheets to a web page.It was developed by Hakon Wium Lie in 1996.</p>
<h2>Java Script</h2>
<p>Java Script is an object oriented programming language. It is used to add dynamic behaviour to a web page. It is also used to validate html forms. It was developed by Brendan Eich in 1995.</p>

function style1()
{
	let element1=document.getElementById("demo");
	element1.style.backgroundColor="yellow";
	element1.style.color="red";
	element1.style.margin="100px";
	let element2=document.getElementById("test");
	element2.align="left";
	element2.style.fontFamily="arial";
	element2.style.fontSize="40px";
	let headings=document.getElementsByTagName("h2");
	for(let heading of headings)
	{
		heading.style.color="black";
		heading.style.fontSize="35px";
	}
	let paragraphs=document.getElementsByTagName("p");
	for(let paragraph of paragraphs)
	{
		paragraph.style.borderStyle="solid";
		paragraph.style.borderWidth="10px";
		paragraph.style.fontSize="30px";
		paragraph.style.padding="30px";
	}
}
