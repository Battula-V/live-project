Attributes of <base> tag:
=====================
1) href="...." => It is used to specify default url. It is applied to 
all links in that particular html document.
2) target="...." => It is used to specify default target. It is applied
to all targets in that particular html document.

Programming Tags:
================
1) <script> .... </script> => It is used to write client side scripting
			code. For example: Java Script code.
2) <noscript>...</noscript> => It is used to specify alternate text.
This text displayed whenever browser does not support script.
3) <style> .... </style> => It is used to write style sheet 
			code(CSS code).

Global Attributes:
===============
Global attributes can be used in html tags.
1) id="..." => It is used to specify unique id for css.
2) class="..." => It is used to specify multiple class names for css.
3) title="..." => It is used to specify tool tip text.
