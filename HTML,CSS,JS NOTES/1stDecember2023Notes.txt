Objects in Java Script:
===================
An object is a collection of properties & methods.

Initially Java Script was an object based & now they added 
new features to Java Script. It is became object oriented.

Object Oriented Language:
=======================
A language that supports all the principles of an object oriented
programming is known as an object oriented language.

Object Oriented Principles(OOPs):
=============================
1) Encapsulation
2) Abstraction 
3) Inheritance
4) Polymorphism

Object Oriented = Object Based + Inheritance + Runtime 
					Polymorphism

Java Script object contains properties & methods.
Properties are variables & Methods are functions.
Java Script object can be referred by references & those 
references are called object references (or) variables.
