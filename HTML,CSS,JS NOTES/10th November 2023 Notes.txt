Data Types:
==========
A data type that determines what value can be stored in
a variable & what are the operations can be performed 
on variable.
In Java Script, there are 8 data types:
1) Number
2) String
3) Boolean
4) Object
5) BigInt
6) Undefined
7) Null
8) Symbol

Java Script is dynamic typed language.
Here data type decided dynamically whenever we initialize
the value to a variable.
It is also called as weakly typed language (or) loosely 
typed language because it does not require to specify the
data type.

Variables:
=========
A variable is a container that contains data.
Here variables can be declared & initialized by using var
keyword.
Recently let & const keywords are introduced to declare
or initialize variables.

Declaration Syntax:
=================
var (or) let (or) const variable-name;

Assignment Syntax:
=================
variable-name=literal;

Initialization Syntax:
=================
var (or) let (or) const variable-name=literal;

Note: Here semicolon(;) is optional.

Java Script Program Structure:
==========================
Java Script code must be written within <script> tags.
<script> tag can be written in <head> or <body> tags.

Example:
========
<html>
<body>
<script>
document.write("Welcome to Java Script");
</script>
</body>
</html>

<!-- Program to demonstrate decimal, octal, hexadecimal & binary literals -->
<html>
<body>
<script>
let a=30;
let b=036;
let c=0x1e;
let d=0b11110;
document.write(a);
document.write(b);
document.write(c);
document.write(d);
</script>
</body>
</html>

Octal literal must be prefixed with 0 (or) 0o (or) 0O.
Hexadecimal literal must be prefixed with 0x (or) 0X.
Binary literal must be prefixed with 0b (or) 0B.
