Arithmetic Operators:
==================
	Operator		Meaning
	========		========
	       +		Addition
	       -		Subtraction
	       *		Multiplication
	       /		Division
	       %		Modulo Division
	      **		Exponentiation

Relational Operators:
==================
	Operator		 Meaning
	========		========
	      <		Less than
	      >		Greater than
	     <=		Less than or equals to
	     >=		Greater than or equals to
	     ==		Equals to
	     !=		Not equals to
	   ===		Strict equals to
			(or) Equal type & equal value
	   !==		Strict Not equals to
			(or) Not equal type & not 
				equal value
The above all operators return boolean type values(true or
false).

<html>
<body>
<script>
let a=5, b="5";
document.write(a==b);
document.write(a!=b);
document.write(a===b);
document.write(a!==b);
</script>
</body>
</html>
