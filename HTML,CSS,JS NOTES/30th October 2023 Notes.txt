Font Properties:
==============
1) font-family => It is used to specify font name.
2) font-size => It is used to specify size of the font.
3) font-style => It is used to specify that italic font or normal font.
4) font-weight => It is used to specify that bold font or normal font.
5) font-variant => It is used to specify that small caps font or 
				normal font.
6) font => It is used to specify the above all property values at a time.
